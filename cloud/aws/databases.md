# Databases

## RDS
- relational db service
- options
  - amazon aurora
    - serverless
  - mysql
  - mariadb
  - prostres
  - oracle
  - mssql
- automating annoying tasks like hardware provisioning and scaling
- fully managed

## DynamoDB
- similar to
  - mongo
  - cassandra
  - oracle nosql
- fully managed

## ElastiCache
- supports
  - redis
  - memcached
- does not support SQL

## Redshift
- data warehouse
- supports SQL
